Software RSS en tu equipo

Otra de las maneras de poder usar RSS consiste en programas instalados en tu equipo los cuales traduces el lenguaje XML para que el usuario pueda ver las noticias de forma legible.
Para ello vamos a usar el programa QuiteRSS que es bastante intuitivo. Como he explicado anteriormente lo primero es suscribirse al canal del que queramos obtener la información, En
este caso vamos a suscribirnos a los dos anteriores.


Marcado como leído

Para marcar todos las noticias como leídas. En este caso en particular el programa nos permite marcar como leído el canal de nuestra elección.


Canal Conciertos

Una vez nos hemos suscrito al canal conciertos (Véase aquí) vamos a proceder a mostrar el primer podcast.

Canal Museo del Prado

Mismo modus operandi para el canal Museo del Prado, lo primero la suscripción (Véase aquí) y a continuacion mostramos la noticia con la imagen o vídeo.
